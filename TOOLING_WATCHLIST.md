# Tooling Watchlist (Back Burner)

Purpose: tools to evaluate at the dedicated content-engine/tooling decision checkpoint, without derailing current MVP execution.

## Queue
1. **Lovart** — https://www.lovart.ai
   - Evaluate: asset quality, editability, consistency, cost/credit model, export usability
2. **Karrot** — https://karrot.ai
   - Evaluate: workflow fit, automation depth, integration options, pricing, reliability

## Evaluation Framework (when we run it)
- Speed to first usable asset
- Cost per accepted asset
- Edit burden to final quality
- Brand consistency across outputs
- Integration friction with DemandOrchestrator
- Reliability / billing / team ops readiness

## Decision Rule
Adopt only if tool materially improves throughput and quality without creating brittle dependencies.
