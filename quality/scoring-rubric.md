# DO Quality Scoring Rubric (Best-of-Breed)

Score each output 1-10 on each axis:

1. Clarity (easy to understand, no jargon fog)
2. Channel Fit (format/tone matches channel)
3. Hook Strength (grabs attention quickly)
4. Conversion Intent (clear CTA or outcome direction)
5. Brand Voice (direct, confident, human)
6. Specificity (not generic filler)
7. Safety/Trust (no spammy or risky claims)

## Pass bar
- Average score >= 8.5
- No axis below 7

## Output review format
- prompt_id
- model_id
- output_text
- axis_scores
- average
- keep_or_rewrite
- rewrite_notes
